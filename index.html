QACanvas

A practice playground for Playwright (TypeScript). Every page targets a specific type of UI element or Playwright feature. Pick a section below to start practicing.

⌨️

Inputs

Text, number, password, range, color, datalist, and more.

 🖱️

Buttons

Click, double-click, right-click, loading, delayed.

 ☑️

Checkboxes

Single, check-all, indeterminate, radio groups.

 📋

Selects

Native, multi, custom, searchable, cascading dropdowns.

 📝

Forms

Full registration form with client-side validation.

 📊

Tables

Static, sortable, filterable, paginated, dynamic tables.

 🔔

Alerts

JS alert/confirm/prompt, custom and nested modals.

 🖼️

Iframes

Simple, form-containing, and nested iframes.

 ↕️

Drag & Drop

Drag between lists, sort, drop into zone.

 🎯

Hover

Tooltips, hover actions, right-click context menus.

 🎹

Keyboard

Key press, shortcuts, tab order, arrow navigation.

 📜

Scroll

Scroll to element, infinite scroll, sticky, tracker.

 ⚡

Dynamic

Appearing, disappearing, skeleton, rotating content.

 🗂️

Tabs

New tab, popup window, cross-tab broadcast.

 👻

Shadow DOM

Simple, nested, and form inside shadow roots.

 🌐

Network

Targets for page.route() interception.

 💾

Storage

localStorage, sessionStorage, cookies — CRUD & persist.
